About This Project

Project Overview

MyToDoList is a task management application built with ASP.NET for the backend and hosted on GitHub Pages for the frontend. User authentication is handled via Entra ID, ensuring secure access. The application stores task data in an Azure SQL Database, supporting CRUD operations (Create, Read, Update, Delete). The ASP.NET API communicates with the database to manage tasks, while the frontend interacts with the backend through API calls. The app allows users to add, update, and delete tasks, with all data securely stored in the cloud.
